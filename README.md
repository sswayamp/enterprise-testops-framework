# Enterprise TestOps Framework

## Overview

This project demonstrates an enterprise-grade UI automation framework built using:

- Java 21
- Selenium 4
- Cucumber BDD
- TestNG
- Docker Selenium Grid
- GitHub Actions
- Jenkins
- Allure Reports
- AI-powered Failure Analysis
